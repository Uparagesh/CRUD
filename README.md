# React CRUD Application

A simple CRUD (Create, Read, Update, Delete) application built using React.js. This project demonstrates API integration, routing, form validation, and state management using React Hooks.

## 🚀 Features

- ✅ Create Record
- ✅ Read Records
- ✅ Update Record
- ✅ Delete Record
- ✅ Form Validation
- ✅ Email Validation
- ✅ Delete Confirmation
- ✅ Loading Spinner
- ✅ Toast Notifications
- 🔄 Search (In Progress)
- 🔄 Sorting (Planned)
- 🔄 Pagination (Planned)

## 🛠️ Tech Stack

- React.js
- React Router DOM
- Axios
- React Toastify
- Bootstrap 5
- MockAPI

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/react-crud-app.git
```

Go to the project folder:

```bash
cd react-crud-app
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
